# Gambo Consultancy - Corporate Website

A professional, academic-grade consultancy website built for **Gambo Consultancy**, specializing in Education, IT, Leadership, Mentorship, and Life Coaching.

This project is a high-performance, responsive frontend application built with the modern Next.js 16+ App Router architecture.

![Gambo Consultancy Logo](/public/logo.jpg)
